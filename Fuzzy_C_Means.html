FUZZY C-MEANS CLUSTERING#

Fuzzy C-Means (FCM) adalah metode clustering berbasis fuzzy logic di mana satu data bisa menjadi anggota lebih dari satu cluster dengan derajat keanggotaan tertentu (nilai antara 0 dan 1). Berbeda dengan K-Means yang bersifat hard clustering (setiap data hanya masuk satu cluster), FCM memperbolehkan overlapping antar cluster.

KONSEP DASAR FUZZY C-MEANS#

  1. Data dapat memiliki keanggotaan parsial ke beberapa cluster.

  2. Setiap data point memiliki nilai derajat keanggotaan (μ) ke setiap cluster.

  3. Tujuan FCM adalah meminimalkan fungsi objektif (cost function) berdasarkan jarak data ke pusat cluster, serta mempertimbangkan nilai keanggotaan fuzzy.

Fungsi Objektif:#

\[ J = \sum_{i=1}^{n} \sum_{j=1}^{c} \mu_{ij}^m \cdot \|x_i - c_j\|^2 \]

\(n\): jumlah data \(c\): jumlah cluster \(\mu_{ij}\): derajat keanggotaan data ke- \(i\) terhadap cluster ke- \(j\) \(x_i\): data ke- \(i\) \(c_j\): pusat cluster ke- \(j\) \(m\): fuzziness parameter, biasanya \(m = 2\)

ALGORITMA FUZZY C-MEANS#

  1. Tentukan jumlah cluster \(c\) dan nilai fuzziness \(m\)

  2. Inisialisasi matriks keanggotaan \(\mu_{ij}\) secara acak (nilai antara 0 dan 1), dengan syarat:

    \[ \sum_{j=1}^{c} \mu_{ij} = 1, \quad \forall i \]

  3. Hitung pusat cluster:

    \[ c_j = \frac{ \sum_{i=1}^{n} \mu_{ij}^m \cdot x_i }{ \sum_{i=1}^{n} \mu_{ij}^m } \]

  4. Perbarui keanggotaan:

    \[ \mu_{ij} = \left( \sum_{k=1}^{c} \left( \frac{ \|x_i - c_j\| }{ \|x_i - c_k\| } \right)^{\frac{2}{m-1}} \right)^{-1} \]

  5. Ulangi langkah 3 dan 4 hingga nilai keanggotaan konvergen (berubah sangat kecil atau mencapai iterasi maksimum).

import numpy as np
import pandas as pd

# Data dan parameter (sesuai data baru)
X = np.array([3, 5, 8, 10], dtype=float)  # Data satu dimensi
c = 2                      # Jumlah cluster
m = 2.0                    # Tingkat fuzziness
epsilon = 0.01             # Ambang batas konvergensi
max_iter = 100             # Maksimum iterasi

# Inisialisasi keanggotaan U secara manual (2 x 4), jumlah tiap kolom = 1
U = np.array([
    [0.7, 0.6, 0.4, 0.2],
    [0.3, 0.4, 0.6, 0.8]
])

# Fungsi untuk menghitung pusat cluster
def calculate_centroids(U, X, m):
    um = U ** m
    return (um @ X) / um.sum(axis=1)

# Fungsi untuk update U
def update_membership(X, V, m):
    n = len(X)
    c = len(V)
    U_new = np.zeros((c, n))
    for i in range(n):
        for j in range(c):
            denom_sum = 0.0
            for k in range(c):
                dist_j = abs(X[i] - V[j])
                dist_k = abs(X[i] - V[k])
                # Hindari pembagian dengan nol
                dist_ratio = dist_j / dist_k if dist_k != 0 else 1e-10
                denom_sum += dist_ratio ** (2 / (m - 1))
            U_new[j, i] = 1.0 / denom_sum
    return U_new

# Fungsi objektif
def objective_function(U, V, X, m):
    obj = 0.0
    for j in range(c):
        for i in range(len(X)):
            obj += (U[j, i] ** m) * (abs(X[i] - V[j]) ** 2)
    return obj

# Iterasi sampai konvergen
history = []
for iteration in range(max_iter):
    V = calculate_centroids(U, X, m)
    obj = objective_function(U, V, X, m)
    history.append((iteration+1, V.copy(), U.copy(), obj))

    U_new = update_membership(X, V, m)

    # Cek konvergensi berdasarkan perubahan centroid
    if np.max(np.abs(V - calculate_centroids(U_new, X, m))) < epsilon:
        print(f"Konvergensi tercapai pada iterasi ke-{iteration+1}")
        break
    U = U_new

# Konversi hasil ke DataFrame agar mudah dibaca
results = []
for iter_num, centroids, membership, obj_val in history:
    result = {
        "Iterasi": iter_num,
        "Centroid 1": centroids[0],
        "Centroid 2": centroids[1],
        "Objektif": obj_val
    }
    for j in range(c):
        for i in range(len(X)):
            result[f"U[{j+1},{i+1}]"] = membership[j, i]
    results.append(result)

df_results = pd.DataFrame(results)
print(df_results.round(4))

# Fuzzy C-Means clustering
n_clusters = 2
cntr, u, u0, d, jm, p, fpc = fuzz.cluster.cmeans(
    data, c=n_clusters, m=2.0, error=0.005, maxiter=1000, init=None, seed=42
)

# Ambil label keras dari keanggotaan tertinggi
labels_2 = np.argmax(u, axis=0)

silhouette_fuzzy = silhouette_score(scaled_features, labels_2)

print(f"[Fuzzy C-Means - 2 Clusters]")
print(f"Silhouette Score: {silhouette_fuzzy:.4f}")
print(f"Fuzzy Partition Coefficient (FPC): {fpc:.4f}")
print(f"Objective Function Value (Jm): {jm[-1]:.4f}")  # ambil nilai terakhir iterasi
print(f"Cluster Centers (in scaled feature space):\n{cntr}")

print("Objective Function Values per Iteration:")
